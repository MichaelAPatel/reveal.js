Templates

Waveforms defined by a set of 63 parameters

  • Mass 1
  • Mass 2
  • Spin X 1
  • Spin X 2

Used to find Gravitational Waves and then for Parameter Estimation

Template Banks

Set of Templates that cover a parameter space

Generated automatically by pyCBC given limits on the parameter space
